## Versioning Strategy

This provider follows [Semantic Versioning](https://semver.org/):

- **MAJOR** version (X.0.0): Breaking changes that require configuration updates
- **MINOR** version (0.X.0): New features and resources in a backwards-compatible manner
- **PATCH** version (0.0.X): Backwards-compatible bug fixes

### Alpha Releases

During the alpha phase (v0.x.x-alpha), the provider is under active development:

- Breaking changes may occur between minor versions
- New resources and features are added frequently
- Bug fixes and improvements are released regularly
- Production use should be carefully evaluated

## Migration Guides

When breaking changes are introduced, detailed migration guides are provided in the release notes. Check the GitHub releases page for version-specific upgrade instructions.
